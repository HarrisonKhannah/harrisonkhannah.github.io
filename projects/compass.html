Website Development, UI Design and Data Visualisation

Work with Compass IoT

Ongoing work with the startup Compass IoT

Compass IoT is a traffic infrastructure platform that can view traffic data on any road, anywhere in Australia, for any time period, instantly.

Through my work on the developing the website and other front-end visuals, alongside working with a team of enginners and marketers - Compass IoT has become a leading industry tool in transport/traffic management projects for councils, governments, consultants and city planners.

Website curation through HTML, CSS and JS

The website for Compass IoT can be seen below. Through the use of HTML and CSS, alongside Javascript and libraries, the website utilities a minimal and sleek UI. Moreover, the website has been attached to an inhouse CRM built by myself. Through all the forms running through "EmailJS" - a clientside JS library that allows forms submissions to be fed directly into the inhouse CRM.

FAQ Site driven by Gatsby.js

Alongside the website was the creation of the FAQ system. This was created through Gatsby.js and utilises Node.js and GraphQL. This is done so the marketing team can curate a FAQ article in Google Sheets and have it instantaneously built on the site through the power of GraphQL. The UI takes a similar approach to the main site and follows the brandings clean, minimal aesthetic

Dynamic data visualisations to monitor traffic flows during COVID-19

A side project in order to display the capabilities of our data was the creation of dynamic data visualisations to monitor the traffic flows during these times. This ongoing release of these visualisations show my work with big data and scripting of JSON alongside working with the Kepler.gl framework.
